# c++的输入和输出

在c++中，输入和输出是使用流来执行的。

输出消息把数据写到输出流中。

# cout c++的标准输出流

cout流是对应者计算机的屏幕的

<<流运算符->用于把数据传递到流中，

cout<< 要输出的内容 << ... << .... <<.. ;

endl 一个换行符

cout<<endl;

cout<<"helloworld"<<endl;

# 变量

数据类型 变量名;

## 数据类型

int 整型 表示的是一定范围内的整数

long 长整型 

bool 布尔型 false true 表示逻辑的真假

char 字符型 用于表示单个字符 'a'

float 浮点型 表示一定范围的小数

double 双精度浮点型 

## 变量名命名规则

①变量名区分字母的大小写，因此B与b表示的是不同的变量。 

②变量名只能由字母、数字和下划线组成，且必须以英文字母开头。例如：b，b1，b1a都是合法的，而1b,_b，b.2，{b}都是不合法的。 

③关键字(如for、end和if等)不能作为变量名。
